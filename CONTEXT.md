# HX Email

本上下文描述注册与验证场景中的邮箱管理领域语言。它关注用户如何组织可用于注册平台、接收验证码和管理绑定关系的邮箱资源。

## Language

**可用邮箱**:
用户可以拿去注册平台、接收验证码、加入邮箱池或绑定用途的具体邮箱地址。主邮箱地址、别名邮箱地址和临时邮箱地址都可以成为可用邮箱；可用邮箱地址在同一用户的数据空间内唯一，不同用户可以各自管理相同邮箱地址。邮箱池领取、验证码读取、平台绑定和前端筛选都以可用邮箱为单位；不再使用的可用邮箱应停用，而不是作为常规操作删除。
_Avoid_: 邮箱账号, 账号, 邮箱

**临时邮箱地址**:
由临时邮箱服务创建或导入的可用邮箱地址。临时邮箱地址没有所属邮箱账号，读信通过临时邮箱 Provider 完成。
_Avoid_: 临时账号

**邮箱账号**:
提供收信能力和凭据配置的容器，例如 Outlook OAuth 账号、普通 IMAP 账号或临时邮箱服务中的邮箱记录。一个邮箱账号可以对应一个主邮箱地址，并可以关联多个已在邮箱服务商后台配置好的别名邮箱地址。
_Avoid_: 可用邮箱, 平台账号

**分组**:
沿用旧版 outlookEmailPlus 的邮箱账号组织方式，用于把邮箱账号按用途或配置归类。分组不是平台绑定，也不是用户权限边界。
_Avoid_: 平台, 用户组

**标签**:
沿用旧版 outlookEmailPlus 的邮箱账号标记方式，用于搜索、筛选和人工整理。标签不是平台，也不表示真实外部绑定关系。
_Avoid_: 平台, 绑定状态

**主邮箱地址**:
邮箱账号默认拥有的原始邮箱地址，是该账号的主要身份标识。主邮箱地址本身也是一个可用邮箱。
_Avoid_: 主账号, 母账号

**别名邮箱地址**:
已在邮箱服务商后台配置、并由某个邮箱账号代收邮件的完整邮箱地址。别名邮箱地址不是 `+tag` 子地址；它本身也是一个可用邮箱，可以独立接收验证码并绑定平台。
_Avoid_: 子账号, 小号, Plus 子地址

**验证码读取**:
为某个可用邮箱查找验证码或验证链接的行为。读取别名邮箱地址或主邮箱地址时，结果必须匹配该可用邮箱的收件地址，不能静默返回同一邮箱账号下其他地址收到的验证码。
_Avoid_: 账号读信

**Plus 子地址**:
由邮箱本地部分追加 `+tag` 形成的派生地址，例如 `user+tag@example.com`。Plus 子地址是旧系统支持的兼容读取能力，不等同于别名邮箱地址。
_Avoid_: 别名邮箱地址

**平台绑定**:
记录某个可用邮箱被用于哪个外部平台或服务的关系。平台绑定的主体是可用邮箱，而不是邮箱账号；一个可用邮箱可以绑定多个平台，一个平台也可以被多个可用邮箱绑定，但同一组可用邮箱与平台只能有一条绑定记录。平台绑定状态包括：正常使用中、待验证、有风险、不可用和已归档。
_Avoid_: 账号绑定, 平台账号

**平台候选**:
系统从邮件发件人、主题或正文中推测出的可能平台。平台候选只用于提示用户，不能自动创建平台或平台绑定。
_Avoid_: 自动绑定

**平台**:
用户用可用邮箱注册或登录的外部网站、应用或服务。平台是可复用的名称实体，不代表某一个具体外部账号；平台名称在同一用户的数据空间内唯一且大小写敏感，前端通过下拉菜单和匹配列表减少重复创建。
_Avoid_: 项目, 分组

**绑定备注**:
平台绑定上的补充说明，用来记录该可用邮箱在该平台上的用途、用户名、限制或其他人工信息。
_Avoid_: 邮箱备注, 平台备注

**用户**:
登录系统并管理自己邮箱资源的人。邮箱账号、可用邮箱、平台、邮箱池和平台绑定都归属于某个用户；不同用户的数据彼此隔离，不引入团队共享概念。
_Avoid_: 调用方, API Key

**管理员**:
拥有用户管理权限的用户。系统通过环境变量提供管理员初始用户名和初始密码，注册默认关闭；管理员可以在系统内修改自己的用户名和密码。
_Avoid_: 团队拥有者

**邮箱池**:
可被自动化注册流程领取、释放、冷却或标记完成的一组可用邮箱。邮箱池条目不拥有邮箱地址本身，只记录某个可用邮箱在自动化任务中的可分配状态。
_Avoid_: 账号池, 号池账号

**临时邮箱 Provider**:
负责创建临时邮箱地址、列出邮件并提取验证码或验证链接的外部服务适配。Provider 是能力来源，不是用户邮箱资源；它创建出来的地址仍以可用邮箱进入工作台。
_Avoid_: 临时邮箱地址, 插件配置

**浏览器扩展任务**:
浏览器扩展围绕邮箱池领取的一次注册辅助流程，包含领取邮箱、等待验证码或验证链接、完成或释放任务。任务不等同于平台绑定；平台绑定需要用户或后端显式记录。
_Avoid_: 平台绑定, 邮箱池条目
