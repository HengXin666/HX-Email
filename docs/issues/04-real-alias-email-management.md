# 为邮箱账号管理真实别名邮箱地址

## What to build

实现真实别名邮箱管理。用户可以为已有邮箱账号录入已在邮箱服务商后台配置好的完整别名邮箱地址；别名邮箱地址作为独立可用邮箱展示、停用和参与后续平台绑定、验证码读取与邮箱池。

## Acceptance criteria

- [ ] 用户可以在添加账号时批量录入别名邮箱地址。
- [ ] 用户可以在账号详情中新增、查看和停用别名邮箱地址。
- [ ] 别名邮箱地址必须是完整邮箱地址，不等同于 `user+tag@example.com` Plus 子地址。
- [ ] 同一用户内主邮箱和别名邮箱地址不能重复，不同用户之间允许重复。
- [ ] 停用某个别名不影响同一账号下其他可用邮箱。
- [ ] 停用邮箱账号会联动使所属别名不可继续作为可用资源使用。
