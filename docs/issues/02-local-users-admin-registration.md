# 实现本地多用户登录、管理员初始化和注册开关

## What to build

实现本地多用户认证闭环。系统通过环境变量提供管理员初始用户名和密码，注册默认关闭；管理员登录后可以修改自己的用户名和密码，并在系统设置中开启或关闭注册。普通用户注册和登录后只能进入自己的数据空间。

## Acceptance criteria

- [ ] 首次启动时可根据 env 初始化管理员用户。
- [ ] 管理员可以登录、退出、修改自己的用户名和密码。
- [ ] 注册默认关闭；管理员可在后台开启或关闭注册。
- [ ] 注册关闭时注册入口隐藏或不可提交，注册 API 返回拒绝结果。
- [ ] 普通用户登录后只能访问自己的业务数据空间。
- [ ] 登录/注册页用 React/Tailwind 重写，视觉参考指定 HX-ANiMe 登录页但不复用 Vue 代码。
