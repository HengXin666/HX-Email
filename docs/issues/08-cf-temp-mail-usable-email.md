# 接入 CF 临时邮箱并纳入可用邮箱工作台

## What to build

实现第一阶段临时邮箱能力，仅支持 CF 临时邮箱。用户可以创建 CF 临时邮箱，系统将其作为没有所属邮箱账号的可用邮箱纳入工作台，并支持读取邮件和验证码。

## Acceptance criteria

- [ ] 用户可以配置并创建 CF 临时邮箱。
- [ ] CF 临时邮箱作为 `temp` 类型可用邮箱出现在工作台。
- [ ] 临时邮箱没有所属邮箱账号，读信通过 CF Provider 完成。
- [ ] 用户可以查看临时邮箱邮件、详情、验证码和验证链接。
- [ ] 临时邮箱支持停用或归档，不影响主邮箱和别名邮箱。
- [ ] 所有临时邮箱数据按用户隔离。
