# 将邮箱池领取状态迁移到可用邮箱

## What to build

实现以可用邮箱为单位的邮箱池。主邮箱、真实别名邮箱和 CF 临时邮箱都可以独立进入邮箱池并被领取；同一邮箱账号下多个可用邮箱允许被不同任务同时领取，读信时仍按可用邮箱地址过滤。

## Acceptance criteria

- [ ] 邮箱池领取、释放、完成和冷却状态挂在可用邮箱上。
- [ ] 主邮箱、别名邮箱和 CF 临时邮箱都可以加入邮箱池。
- [ ] 同一邮箱账号下多个可用邮箱允许同时被不同任务领取。
- [ ] `project_key` 成功复用语义按可用邮箱计算。
- [ ] 领取后的验证码读取使用被领取可用邮箱的收件地址过滤。
- [ ] 邮箱池数据和统计按用户隔离。
