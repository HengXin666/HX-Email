# 按可用邮箱收件地址读取验证码和验证链接

## What to build

实现以可用邮箱为单位的验证码读取。用户从主邮箱或真实别名邮箱触发读取时，系统通过所属邮箱账号收信，但必须按目标可用邮箱的收件地址过滤邮件，避免同一账号下不同地址的验证码串用。

## Acceptance criteria

- [ ] 用户可以从可用邮箱详情触发验证码和验证链接读取。
- [ ] 主邮箱读取只返回匹配主邮箱收件地址的验证码邮件。
- [ ] 别名邮箱读取只返回匹配该别名收件地址的验证码邮件。
- [ ] 如果上游缺少可靠收件地址，结果必须标记为不确定或提示确认，不能静默当作确定命中。
- [ ] 旧版 Plus 子地址兼容读取能力可保留，但不能替代真实别名邮箱逻辑。
- [ ] 读取历史按用户和可用邮箱隔离。
