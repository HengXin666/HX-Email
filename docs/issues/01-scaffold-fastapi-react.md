# 搭建 FastAPI + uv 与 React + shadcn/ui 重写骨架

## What to build

搭建 HX Email 重写项目的可运行骨架。后端使用 FastAPI 与 uv，前端使用 React、TypeScript、shadcn/ui 与 Tailwind，默认黑夜主题。项目启动后应能完成基础健康检查、前端路由加载和测试命令运行，为后续垂直切片提供稳定入口。

## Acceptance criteria

- [ ] 后端可通过 uv 启动 FastAPI，并提供健康检查接口。
- [ ] 前端可启动 React + TypeScript + shadcn/ui + Tailwind 应用，默认黑夜主题。
- [ ] SQLite 数据目录、配置加载和迁移入口具备最小可运行形态。
- [ ] 前后端测试命令存在并能在空功能骨架上通过。
- [ ] README 或开发文档说明本地启动、测试和配置方式。

## Blocked by

None - can start immediately
